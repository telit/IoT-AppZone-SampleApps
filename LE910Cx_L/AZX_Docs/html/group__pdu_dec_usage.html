pdu_struct *  packet,

CHAR *  msg 
)

Decodes an incoming PDU in binary format.

This function decodes the incoming PDU (in binary stream format) to output structure. Sender Number and SMS text message are returned as strings

Parameters
[in]pduPointer to the buffer containing the pdu stream in binary format
[in]pdu_lenSize in bytes of the pdu
[in]packetPointer to the pdu_struct that will be filled with all the information about the PDU
[in]numberString buffer that will be filled with the caller number
[in]msgString buffer that will be filled with the incoming text message in ascii format
Returns
The length of the decoded text message

Refer to azx_pdu_Encoder()


Examples
azx_pduDec_example.c.

UINT16  in_len 
)

Converts UTF8 spaces into ASCII spaces.

If the decoded PDU has an UTF8 coding, spaces will be returned as 2 bytes (0xc2, 0xa0), this function converts them to ASCII spaces (0x20)

Parameters
[in,out]inpointer to the buffer containing the UTF8 data. It will be converted and in param will be overwritten
[in]in_lenlength of the buffer
Returns
The length of the converted message
