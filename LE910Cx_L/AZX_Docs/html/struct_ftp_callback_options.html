Detailed Description

Structure that holds status callback function parameters.

This structure holds all the parameters that can be used to set a transfer status callback. It will be called by the client each time one of the set event occurs (timeout or bytes transferred)

Refer to azx_ftp_setCallback() azx_ftp_clearCallback()
