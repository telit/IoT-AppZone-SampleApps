Detailed Description

Configuration functions, macros and enumerators can be used to set the logging channel (Main UART, Auxiliary UART, USB0/ USB1) and the logging mask (which allows to enable each log level independently)

enum AZX_LOG_LEVEL_E

Logging levels.

Enumerator
AZX_LOG_LEVEL_TRACE 

Prints every message, adds a "TRACE" at the beginning of the message

AZX_LOG_LEVEL_DEBUG 

Prints most of the messages, adds a "DEBUG" at the beginning of the message

AZX_LOG_LEVEL_INFO 

print the message only, without any additional info

AZX_LOG_LEVEL_WARN 

like DEBUG, but adds a "WARNING" at the beginning of the message.

AZX_LOG_LEVEL_ERROR 

like DEBUG, but adds an "ERROR" at the beginning of the message

AZX_LOG_LEVEL_CRITICAL 

like DEBUG, but adds an "CRITICAL" at the beginning of the message

AZX_LOG_LEVEL_NONE 

Do not print anything

◆ azx_log_init()

void azx_log_init ( AZX_LOG_CFG_T *  cfg)

Initializes the log functionality.

This function initializes the log functionality with the provided configuration structure. Call this before any AZX_LOG_* are used.

It is recommended to use the macro AZX_LOG_INIT.

Parameters
[in]cfgPointer to the structure with the configuration parameters
Returns
None

Sample usage 

AZX_LOG_CFG_T cfg = {
   .log_level = AZX_LOG_LEVEL_DEBUG,
   .log_channel = AZX_LOG_TO_MAIN_UART,
   .log_colours = FALSE
};

azx_log_init(&cfg);  //initialize log system
