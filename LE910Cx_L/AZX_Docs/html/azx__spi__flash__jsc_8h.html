Contains the Software Drivers for SPI NAND FLASH JSC JS28(P_U)xGQSxHG-83 family.

anonymous enum

oob operation modes

@MTD_OPS_PLACE_OOB: OOB data are placed at the given offset (default) @MTD_OPS_AUTO_OOB: OOB data are automatically placed at the free areas which are defined by the internal ecclayout @MTD_OPS_RAW: data are transferred as-is, with no error correction; this mode implies MTD_OPS_PLACE_OOB

azx_spi_flash_blockIsBad checks block is bad or not

Checks block is bad or not via read

Note
bad block mark(the first two byte in oob area of the first page in the block) For a block, read the first page's first two byte of oob data, if data is all 0xFF, the block is a good block, otherwise it a bad block.
Parameters
[in]chipAZX_SPI_FLASH_SPI_CHIP *chip: SPI-NAND device structure
[in]offsJSC_loff_t offs: offset from device start
[in]resultJSC_bool *result: true if is bad block, false otherwise
Returns
azx_spi_flash_blockIsBad() returns AZX_SPI_FLASH_CODE_PASS if there isn't error. Otherwise it returns AZX_SPI_FLASH_CODE_RESULT_E. If success result is true if is bad block, false otherwise
