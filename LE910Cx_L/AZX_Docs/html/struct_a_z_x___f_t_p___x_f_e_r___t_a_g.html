Detailed Description

Generic transfer information structure.

This structure holds the information what kind of transfer media to be used for the ftp get or put operations. It can be file or buffer then the payload can be either a AZX_FTP_FILE_INFO or AZX_FTP_BUFFER_INFO structure

Refer to azx_ftp_get() azx_ftp_put()

Sample usage

char buffer[1500];
AZX_FTP_BUFFER b = { buffer, 1500, my_data_cb};
AZX_FTP_XFER x;
x.type = AZX_FTP_XFER_BUFF;
x.payload = b;
