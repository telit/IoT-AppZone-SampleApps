28 /* closesocket() is a windows function. Linux uses the close() function for both

31  and M2MB_T_FS_HANDLE is a INT32
32  Therefore look for all the close() calls in your code and rename to closesocket()
33  the ones that refer to sockets, or find some other way around it.
