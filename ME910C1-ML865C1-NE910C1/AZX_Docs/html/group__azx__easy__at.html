typedef void( * azx_easy_at_taskCallback) (M2MB_ATP_HANDLE h, UINT16 atpI)

This is the custom command easy at main callback function signature.

This callback function is executed by ATP when a custom AT command is executed. This will contain user code to manage the command

Parameters
[in]hthe ATP handle associated with the command
[in]atpIthe atp instance where the command was received. Can be used internally with m2mb_atp_get_input_data to retrieve the command information (name, parameters list and so on)
Returns
None

typedef void( * azx_easy_at_taskDelegation) (M2MB_ATP_HANDLE h, UINT16 atpI, M2MB_ATP_DELEGATION_IND_E delegationEvent, UINT16 msg_size, void *delegationMsg)

This is the custom command easy at delegation callback function signature.

This callback function is executed by ATP when a custom AT command is in execution, and user data input is expected. This will contain user code to manage the command

Parameters
[in]hthe ATP handle associated with the command
[in]atpIthe atp instance where the command was received. Can be used internally with m2mb_atp_get_input_data to retrieve the command information (name, parameters list and so on)
[in]delegationEventthe delegation event (can be DATA or ESCAPE)
[in]delegationMsgSizeadditional delegation callback data size in bytes
[in]delegationMsgadditional delegation callback data pointer
Returns
None

M2MB_RESULT_E azx_easy_at_CMEE ( AZX_EASY_AT_HANDLES_T *  hdls,
INT16  cmee_code,
CHAR *  cust_cmee_message,
  ... 
)

function to output an error message over ATP interface.

This function will output an error message on ATP interface. The format of the message depends on the cmee level for that interface. Custom cme code and message can be provided. It is internally called with the macro AZX_EASY_AT_RELEASE_WITH_CMEE

Parameters
[in]hdlspointer to the structure with atp handle and atp instance info
[in]cmee_codethe numeric value of the error. can be one of the standard
[in]levelthe custom message, if needed. Leave to NULL to use a standard message. It Supports variadic parameters
Returns
M2MB_RESULT_E value
