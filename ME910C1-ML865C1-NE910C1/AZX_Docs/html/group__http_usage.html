◆ azx_http_get()

int azx_http_get ( AZX_HTTP_INFO *  hi,
char *  url 
)

Run a HTTP GET request.

Run a HTTP GET request using the URL info and the additional info stored in HTTP_INFO info. It supports single RANGE requests.

Parameters
[in,out]hiAZX_HTTP_INFO struct used to provide additional data for the request (i.e the HTTP authentication type) and store the retrieved metadata from the header
[in]urlthe URL to parse for the HTTP request
Returns
HTTP response code - Ok
-1 - Failure

Refer to azx_http_initialize() azx_http_SSLInit() azx_http_set_CB()

Sample usage

AZX_HTTP_INFO hi;
char respRange[100] = {0};
memset(&hi, 0, sizeof(HTTP_INFO));
hi.request.auth_type = AuthSchemaBasic;
hi.request.content_range = (char *)"0-50";
hi.response.content_range = respRange;
azx_http_get(&hi,"http://guest:guest@jigsaw.w3.org/HTTP/Basic/");
Examples
azx_https_example.c.

Run a HTTP POST request.

Run a HTTP POST request using the URL info and the additional info stored in HTTP_INFO info

Parameters
[in,out]hiAZX_HTTP_INFO struct used to provide additional data for the request (i.e the POST data) and its content_length and store the retrieved metadata from the header
[in]urlthe URL to parse for the HTTP request
Returns
HTTP response code - Ok
-1 - Failure

Refer to azx_http_initialize() azx_http_SSLInit() azx_http_set_CB()

Sample usage

AZX_HTTP_INFO hi;
memset(&hi, 0, sizeof(AZX_HTTP_INFO));
char *data = "This is the POST data";
hi.request.post_data = (char *)m2mb_os_malloc(sizeof(char)* (strlen(data) + 1));
memset(hi.request.post_data, 0, sizeof(sizeof(char)* (strlen(data) + 1)));
strcpy(hi.request.post_data,data);
hi.request.content_length = strlen(hi.request.post_data);
ret = azx_http_post(&hi,(char *)"http://postman-echo.com:80/post");
