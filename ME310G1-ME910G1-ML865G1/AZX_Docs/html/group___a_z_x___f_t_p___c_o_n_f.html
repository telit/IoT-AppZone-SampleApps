◆ azx_ftp_callback

typedef int(* azx_ftp_callback) (AZX_FTP_NET_BUF_T *nControl, azx_ftp_fsz_t xfered, void *arg)

Callback function signature.

This is the callback function signature (optional) that the client will call during data exchange with the remote server. User callback must follow this format.

Parameters
[in]nControlPointer to the FTP client structure pointer
[in]xferedIndicates how much data bytes have been transferred until now
[in]argUser parameter, set in azx_ftp_setCallback()
Returns
1 if the FTP client must keep running
0 to force the FTP client to interrupt the current operation and close the data socket.

Refer to azx_ftp_setCallback() azx_ftp_clearCallback()

Sample usage

AZX_FTP_NET_BUF_T *ftp_client;
FtpCallbackOptions opt;
opt.cbFunc = log_progress;
opt.cbArg = &f;
opt.idleTime = 1000; //call each second
opt.bytesXferred = 1024; //call each 1024 bytes
azx_ftp_setCallback(&opt,ftp_client);

◆ azx_ftp_debugHook

typedef INT32(* azx_ftp_debugHook) (AZX_FTP_DEBUG_HOOK_LEVELS_E level, const CHAR *function, const CHAR *file, INT32 line, const CHAR *fmt,...)

Debug callback that is called by the library to for debug purposes.

This function prototype can be registered by the user with the azx_ftp_init call. It is user's responsibility to define the logic of all debug levels (see FTP_DEBUG_HOOK_LEVELS_E enum above ) and how they are printed.

Parameters
[in]levelDebug level, see FTP_DEBUG_HOOK_LEVELS
[in]functionThis indicates the function name where the debug print is occurring
[in]fileThis indicates the .c file name where the debug print is occurring
[in]lineThis indicates the .c file line number where the debug print is occurring
[in]fmtformat string (as in printf). It contains the log message to be printed, plus possible additional parameters, to be used with va_list.
Returns
An integer value, not used at the moment

Refer to azx_ftp_setCallback() azx_ftp_clearCallback()

Sample usage

#include <stdarg.h>
int (FTP_DEBUG_HOOK_LEVELS_E level, const char *function, const char *file, int line, const char *fmt, ...)
{
char buf[512];
int bufSize = sizeof(buf);
va_list arg;
int offset = 0;
memset(buf,0,bufSize);
switch(level)
{
case FTP_DEBUG_HOOK_ERROR:
offset = sprintf(buf, "ERR %32s:%-4d - ",
function,
line
);
break;
case FTP_DEBUG_HOOK_INFO:
offset = 0;
break;
case FTP_DEBUG_HOOK_DEBUG:
offset = sprintf(buf, "DBG %32s:%-4d - ",
function,
line
);
break;
}
va_start(arg, fmt);
vsnprintf(buf + offset, bufSize-offset, fmt, arg);
va_end(arg);
return PRINT(buf); //NOTE: PRINT is a generic print function like printf
}
