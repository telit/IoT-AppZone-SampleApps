#define AZX_LIMIT (   val,
  min,
  max 
)    (val = (val < (min) ? (min) : (val > (max) ? (max) : val)))

Limits a value to a certain interval.

This is defined as a macro so it works with any types that support ordering.

Parameters
valThe value to limit
minThe minimum allowed value
maxThe maximum allowed value
Returns
The original value if it was inside the limits, or min, or max, if it was less than, or greater than the limits.
