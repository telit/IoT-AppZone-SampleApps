Detailed Description

Generic buffer information structure.

This structure holds the information of a buffer, where data can be stored during a file get or from where data can be retrieved during a file put.

It contains

the buffer size

Refer to AZX_FTP_XFER azx_ftp_get() azx_ftp_put()

Sample usage

char buffer[1500];
AZX_FTP_BUFFER b = { buffer, 1500, my_data_cb};
